# ES6 Basics

This project contains JavaScript ES6 practice tasks.

## Learning Objectives

- ES6 basics
- `const` and `let`
- Block scope
- Arrow functions
- Default parameters
- Rest parameters
- Spread syntax
- Template literals
- Object property shorthand

## Requirements

- Ubuntu 20.04
- Node.js 20.x.x
- npm 9.x.x
- Jest
- Babel
- ESLint
